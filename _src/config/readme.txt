
# Environment Variables (ENV) are available from the code via [process.env.MY_VARIABLE]
  * for the development version of the project, ENV are stored in "/config/.env" file and connected to the project via "dotenv" plugin in package.json (scripts/dev)
  * for the product version, ENV are connected according to the product environment rules (depends on the environment)
